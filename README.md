## Featured projects

**[IDS-Platform](https://github.com/Genti411/IDS-Platform) - Enterprise Intrusion Detection System**
Full-stack IDS: a Python detection engine (packet capture, rules, anomaly detection), a FastAPI + WebSocket API, and a React/TypeScript SOC dashboard. Threat-intel enrichment, Kafka event streaming, PostgreSQL + OpenSearch + Redis, containerized with Docker/Kubernetes and provisioned via Terraform.

**[Red-Team-Security-Assessment](https://github.com/Genti411/Red-Team-Security-Assessment) - Red Team Framework**
Modular APT-simulation framework with a custom exploit launcher, command-and-control server/agent, social-engineering payloads, and auto-generated post-exploitation reports in a Dockerized lab.

**[Automated-Vulnerability-Scanner](https://github.com/Genti411/Automated-Vulnerability-Scanner) - Web App Vulnerability Scanner**
Detects XSS, SQL injection, and CSRF with a focus on low false positives. OWASP Top 10 / NIST mapping, severity-based prioritization, JSON/HTML reports, a Flask dashboard, and CI/CD integration.

**[Network-Security-Monitoring-System](https://github.com/Genti411/Network-Security-Monitoring-System) - ML Network Monitor**
Captures live network traffic and flags anomalies using Isolation Forest, with logging, alerting, and a Flask dashboard. Deployable via Docker or AWS EC2.

**[Zero-Trust-Architecture-Implementation](https://github.com/Genti411/Zero-Trust-Architecture-Implementation) - Zero Trust Framework**
End-to-end ZTA: RBAC/ABAC access control, automated compliance checks (MFA enforcement, role validation), IoT posture audits, Terraform provisioning, and Grafana/Kibana dashboards.

**[Login-Anomaly-Detector](https://github.com/Genti411/Login-Anomaly-Detector) - Login Anomaly Detection**
Flags anomalous login activity from auth logs using Isolation Forest and autoencoder models, served through Flask APIs.
